pure JavaScript content locker
What is content locker?
Content Locker is a simple and straitforward solution for monetizing of web sites. Content locking easely generates more profit than PPC and display advertising models. It can also be used to emphasize user attention on some features of the web site and the neccessary actions that the user must perform to get them.

When a visitors comes to a content locked page, they are prompted to complete one of the actions (for example - customer surveys). Once the action is complete, you are recieve your benefit and the visitor gets access to you site's content.
What content can be locked?
  • How to sites (building designs, potato gun, skateboard ramp)
  • Music lessons (video, guide, chords, etc)
  • Graphics (photos, images, icons, designs, patterns, etc)
  • Games (Flash, online games, etc)
  • Templates (Word-press, Joomla, Drupal, etc)
  • Programming (HTML, PHP, JavaScript, C++, etc)
  • Game cheats (PS3, Xbox, PC, etc)
  • Ebooks (how to, instructional, guides, tutorials, etc)
  • Alternative check out (complete an offer to checkout, etc)
  • Doc/HTML files to PDF files (have users complete an offer in order to have there files converted)
  • Membership upgrades (allow users to complete offer to upgrade a forum membership)
  • Quality Content (blogs, quotes, posts, etc)
  • Musicians (instead of charging money for your music, have users complete an offer)
  • Android/Apple Apps (monetize your apps, with content locking to an upgrade, or allowing users to complete an offer to gain more in game chips/tokens)
You must own the copyrights to all content you lock!
What can be customized?
  • LoadEvent - event which will trigger the content locker
  • ElementPattern - or CSS pattern, which will be used along with LoadEvent option
  • ContentAttachment - how content will be attached to the locker
  • ContentHAlign - content horizontal align
  • ContentVAlign - content vertical align
  • Width - width of the locker
  • Height - height of the locker
  • ContentWidth - width of the content
  • ContentHeight - height of the content
  • ContentPadding - content padding
  • FitBorderImage - width and height of the border will fit (or not) border image
  • OverlayImageURL - overlay image URL
  • BorderImageURL - border image URL
  • OVOpacity - overlay opacity
  • BGOpacity - border opacity
  • OVColor - overlay color (if OverlayImageURL is not set)
  • BGColor - locker background color
  • ContentURL - URL from which offers will be loaded
  • StatusCheckURL - URL to which "status check" requests will be made
  • ShowDelay - delay before locker will be displayed
  • CheckStatusDelay - delay before every "status check" request

Setup
1.1. Download Electro content locker form here
2.1. Copy locker.js to the folder where all of your JavaScript files resides
3.1. Link to locker.js from your html file:

<script type="text/javascript" src="[PATH]/locker.js"></script>

or

1.2. Link to locker.js from your html file using external file:

<script type="text/javascript" src="https://raw.github.com/Error-331/Electro-content-locker/master/locker/locker.js"></script>

4. Configure content locker, example:

var usrConfig =
{
LoadEvent: 'onclick',
ElementPattern: '',

ContentAttachment: 'normal',
ContentHAlign: 'left',
ContentVAlign: 'top',

Width: 500,
Height: 350,
ContentWidth: 450,
ContentHeight: 300,

ContentPadding: 0,

FitBorderImage: false,

OverlayImageURL: 'img/bg/bg8.gif',
BorderImageURL: 'img/border/brd1.png',

OVOpacity: 100,
BGOpacity: 100,

OVColor: '#FF7F24',
BGColor: '#FFFFFF',

ContentURL: 'content.txt',
StatusCheckURL: 'status.txt',

ShowDelay: 0,
CheckStatusDelay: 5000
};

5. Setup content callback (URL path set in 'ContentURL' configuration option):

var tmpContent = '<div class="DemoLockContOut">';
tmpContent += '<h3 class="DemoLockHead">Content locker</h3>';
tmpContent += '<h4 class="DemoLockDesc">Please complete one of the following offers</h4>';
tmpContent += '<ul class="DemoLockList">';
tmpContent += '<li><a href="#">Offer long name 1</a></li>';
tmpContent += '<li><a href="#">Offer long name 2</a></li>';
tmpContent += '<li><a href="#">Offer long name 3</a></li>';
tmpContent += '<li><a href="#">Offer long name 4</a></li>';
tmpContent += '</ul>';
tmpContent += '</div>';
tmpContent += '<br/>';
tmpContent += '<span class="DemoLockStatStr">Status checking...</span>';
tmpContLock.SetContent(tmpContent);

6. Setup status check callback (URL path set in 'StatusCheckURL' configuration option):

..Some code
..
..
tmpContLock.AddContent('<span style="color: green;">.</span>');
tmpContLock.StartStatusChecking();
..
..
..or stop status checking if some condition is met
..
..
tmpContLock.HideLocker();

7. Initiate and show content locker:

..Some code
..
tmpContLock = new ElectroContentLocker(usrConfig);
..
